# Dynamic Pricing Optimization Model

## 📊 Project Overview
This project demonstrates how **dynamic pricing** can maximize profit in ride-sharing using:
- Machine Learning (Random Forest, Linear Regression)
- Optimization formula: (Price * Demand) - Cost
- Power BI dashboard for visualization

---

## 🔧 Key Features
- Data Cleaning: handled missing values, outliers, normalized ride cost
- Exploratory Analysis: demand vs cost, loyalty vs rides, category-wise revenue
- Feature Engineering: cost buckets, loyalty flags, demand indicators
- Model Building: Random Forest & Linear Regression
- Optimization: profit curve with optimal price point
- Dashboard: KPIs + charts (donut, pie, bar, funnel, line)

---

## 🖼️ Dashboard Snapshots
- Total Rides, Past Rides, Average Ratings, Revenue (KPI cards)
- Revenue share by vehicle type
- Demand trend by time of booking
- Loyalty vs revenue comparison

---

## 📈 Tech Stack
- **Python**: Scikit-learn, Pandas, NumPy, Matplotlib
- **SQL**: Advanced queries for revenue & demand analysis
- **Power BI**: DAX, Power Query, KPI cards, interactive charts
